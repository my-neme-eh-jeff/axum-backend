Download and Install Rust:
  ```bash
  curl https://sh.rustup.rs -sSf | sh
  ```

Verify the Installation:
  ```bash
  rustc --version
  ```

Clone the Repository:
  ```bash
  git clone https://github.com/my-neme-eh-jeff/movie_listing_filtering.git
  ```

Run the Project:
  ```bash
  cargo run
  ```
